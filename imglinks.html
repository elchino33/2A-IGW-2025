USO DE ETIQUETAS MULTIMEDIA Y ENLACES EN HTML

etiqueta <img>

se usa para insertar imagenes. es uma etiqueta vacia, no tiene cierre.

etiqueta <video>

permite insertar videos. se recomienda usar varios formatos para compatibilidad.

video de yutube

para insertar u video de yutube, usa la etiqueta <iframe>.

feed de instagram

para insertar un feed de imstagram, usa la etiqueta <blockquote>

etiqueta <a>

sirve para crear enlaces a otras paginas, documentos o ubicaciones.
